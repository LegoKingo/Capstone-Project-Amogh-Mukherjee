**Known Issues:**
Right now there are a few text encoding errors spread throughout the game that I simply couldn't eliminate in time. There are a few apostrophes and quotation marks that became unknown characters.

The player UI is occasionally drawn behind objects in the game level, I wasn't able to figure out what consistently caused this, so I couldn't fix it.

Boost ability is very glitchy when used too much. Movement gets very unpredictable and it's very hard to slow down. Be careful using it!
